
# Portfolio Web

## Bienvenido a mi portfolio personal

Este refleja mi aprendizaje continuo y mi interés por el desarrollo web. Se irá actualizando constantemente a medida que mejore los proyectos existentes o cree otros nuevos. [Ver sitio](https://facundo-daddese.netlify.app/)

## ¿Qué contiene?

- Mis habilidades y conocimientos relacionadas con el FrontEnd.  
- Los proyectos en los que estoy trabajando o que ya completé.
- Un formulario de contacto, desde el cual cualquier mensaje enviado llega directamente a mi email.
- Enlaces a mis redes y a mi CV.

## Tecnologías y habilidades

- HTML5, CSS3, JavaScript(ES6+)
- Tailwind CSS
- React
